# 🧑‍🤝‍🧑 KeenKeeper — Keep Your Friendships Alive

## Description
KeenKeeper is a friendship management web application designed to help users stay connected with their friends. It allows tracking of interactions, setting relationship goals, and visualizing communication patterns. The app emphasizes responsive design, smooth navigation, and meaningful data presentation to keep friendships alive and thriving.

## Technologies Used
- **React.js / Next.js** — Build the user interface  
- **React Router DOM / Next.js App Router** — Handle page navigation  
- **Tailwind CSS + Component Library** — Styling and responsiveness  
- **Recharts** — Data visualization (charts and analytics)  

## 3 Key Features
1. **Friend Management** — Display friends as interactive cards with profile details, tags, and status indicators.  
2. **Timeline Tracking** — Log and view all interactions (calls, texts, video calls) with toast notifications.  
3. **Friendship Analytics** — Visualize communication patterns using charts, including counts of different interaction types.
